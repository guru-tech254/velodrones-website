# Velodrones Website

Premium aerial intelligence and geospatial solutions website.

## Setup

1. Install Sanity CLI: `npm install -g @sanity/cli`
2. Initialize Sanity: `sanity init`
3. Replace `YOUR_SANITY_PROJECT_ID` in `js/portfolio.js` and `js/sanity-client.js`
4. Replace `YOUR_FORMSPREE_ID` in `contact.html`
5. Add images to `assets/images/` folder
6. Add video to `assets/hero-video.mp4`

## Deploy

Push to GitHub and connect to Vercel/Netlify.
